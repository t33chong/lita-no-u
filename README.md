# lita-no-u

**lita-no-u** is a handler for [Lita](https://github.com/jimmycuadra/lita) that replies with 'no u'.

## Installation

Add lita-no-u to your Lita instance's Gemfile:

``` ruby
gem "lita-no-u"
```

## Usage

To trigger the response 'no u':

```
You: no u
Lita: no u
```
